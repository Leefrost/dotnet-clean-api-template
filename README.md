## About
A clean template based on ASP.NET CORE Web API, CQRS and Clean Architecture workflow. Allow you to create simple app without any pain regarding solution structure or dependencies.

Stack:
- ASP.NET CORE / 5.0
- OPEN Api specification
- CQRS (MediatR)
- Tests (NUnit)
- Nuget
